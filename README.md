# CRO Audit Tool

Een standalone HTML tool voor het uitvoeren van CRO (Conversion Rate Optimisation) audits op webshops. Geen installatie, geen server — gewoon het `.html` bestand openen in je browser.

## Gebruik

1. Download `cro-audit.html`
2. Open in je browser
3. Vul klant, URL, datum en auteur in
4. Doorloop de 50 checkpoints verdeeld over 10 categorieën
5. Exporteer het rapport als HTML of tekst

## Categorieën

| # | Categorie | Checkpoints |
|---|-----------|-------------|
| 01 | Homepage | 8 |
| 02 | Menu | 8 |
| 03 | Zoeken | 4 |
| 04 | Overzichtspagina's & filtering | 6 |
| 05 | Product detail pagina | 6 |
| 06 | Checkout | 9 |
| 07 | Unique selling points | 3 |
| 08 | Call-to-actions | 5 |
| 09 | Navigatie & informatie | 6 |
| 10 | Algehele gebruikerservaring | 4 |

## Scoretypen

- ✓ **Geen actie vereist** — onderdeel scoort goed
- ◐ **Verbeterkans** — ruimte voor optimalisatie
- ⚑ **Noodzakelijk actiepunt** — moet worden opgepakt

## Features

- Live voortgangsbalk en scoretellers
- Notitieveld per checkpoint
- Sidebar navigatie met status per sectie
- Export naar HTML-rapport (klaar voor klant) of platte tekst
